tmux-pane-rel
=============
A helper for quickly resizing tmux panes to a percentage of a session's total
width/height

Here's what it looks like:
![tmux-pane-rel-demo](/tmux-pane-rel-demo.gif)

## Usage
```bash
tmux-pane-rel --axis <axis=x|y> <percentage>
```
